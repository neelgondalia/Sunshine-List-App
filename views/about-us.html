About The Tool

This application was developed as a quick project for a course at the University of Guelph. The goal of this application is to allow a user to analyze publically available data on high earners in the public sector.

About the data

The data for this project has been sourced from the following sites:

  • Ontario.ca
  • Alberta.ca
We do not guarantee the accuracy of the data.
